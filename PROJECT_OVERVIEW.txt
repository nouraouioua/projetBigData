FONCTIONNALITÉS PRINCIPALES
══════════════════════════════════════════════════════════════════════

[OK] Parsing Avancé
   • Regex complexe pour logs Apache
   • IP, timestamp, URL, méthode, status, bytes
   • Validation et enrichissement

[OK] Analyses SQL
   • Accès par heure/minute
   • Top 20 URLs et IPs
   • Distribution codes HTTP
   • Analyse des erreurs
   • Détection des pics

[OK] Machine Learning (MLlib)
   • K-Means clustering
   • Détection statistique (Z-score)
   • Identification IPs suspectes
   • Feature engineering

[OK] Analyse de Graphe (GraphX)
   • Graphe biparti IP <-> URL
   • Connectivité et popularité
   • Community detection
   • Patterns suspects

[OK] Spark Streaming
   • Structured Streaming
   • Simulation temps réel (10s)
   • Multiple outputs
   • Watermarking

[OK] Dashboard Grafana
   • Access count per minute
   • Top endpoints
   • HTTP status distribution
   • Anomalies détectées
   • IPs suspectes


DÉMARRAGE RAPIDE
══════════════════════════════════════════════════════════════════════

1.  Télécharger les données
    ./download_data.sh

2.  Installer les dépendances
    pip install -r requirements.txt

3.  Lancer l'analyse
    python main.py
    # OU
    ./run.sh
